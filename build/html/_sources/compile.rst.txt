Install and Run the CHIPS Pangea Poker
======================================

In order to run the project you need to install and run the following in the given order:

1. Install CHIPS-cli wallet

2. Install Lightning Network node. Make sure to install CHIPS-cli wallet first, the CHIPS daemon should be synced!

3. Next to setting up the CHIPS daemon and putting a node up in the Lightning Network, BET is needed to bridge the two together.

Please find the compilation and installation instructions at the `Bet Repo <https://github.com/chips-blockchain/bet/blob/master/compile.md>`_

You can also use `Docker container <https://github.com/chips-blockchain/bet/blob/master/compile.md>`_ with all the dependencies pre installed and compiled for you.
